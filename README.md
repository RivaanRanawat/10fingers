# 10FINGERS

## Features

* Game Menu
* Creating Game
* Joining Game
* Verification Before Joining
* Private Rooms
* Room Creator Button To Start Game
* Timer to Get Ready
* Game Clock to Type
* Highlighting Typed Words
* Underlining Next Word
* Showing Words to Be Typed
* Progress Bar Showing % of Words Typed
* Displaying WPM
* Copy Game Code Button

## Technologies Used

1. MongoDB
2. Mongoose
3. Express
4. Node
5. React
6. React Router DOM
7. Sockets
